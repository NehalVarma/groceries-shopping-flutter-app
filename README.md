### Installation

<img src="screenshots/screenshot10_new.png" width="250"> <img src="screenshots/screenshot11_new.png" width="250">

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/groceries-shopping-app.git
cd groceries-shopping-app
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```
